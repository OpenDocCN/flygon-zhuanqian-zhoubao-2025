## (60 赞)系统性学习某方面知识的建议

日期：2025-10-09

提问：

亦仁老师你好，目前想要进入 web3 这个圈子，但了解的方式太匮乏了，目前通过 x 上搜罗各种信息来了解都比较杂乱。想请问一下有没有什么书籍推荐，能更好的系统性学习这方面的知识。

回答

没什么推荐的书，而且 web3 赚钱和你看书了解区块链是两码事。 想快速了解，可以加入几个付费圈子，顺着这些社群的脉络去了解，可以少踩一些坑。
当然，哪些新圈子好，哪些不好，得自己判断。我进入新领域，会默认有至少一半的学习费用是打水漂的，也就是毫无价值，它们的价值就是为了让我筛选出好的产品。
不过有一些生财老圈友，比如紫菜，老薛，金马，也有付费社群，你可以找过去问问看。
如果一个产品存在的时间足够长，那大概率他们不会轻易犯错，毕竟他们的犯错成本要比一个新的产品高太多了，时间才是最大的成本，多年积累下来的信誉不值得轻易犯错。
就像生财有术再过一两年就是一个“十年老店”了，这本身就是一个巨大的信任了，我犯不着去做一些割韭菜的事情，对吧？
