做副业并不一定需要整块时间 做副业并不一定需要整块时间 做副业并不一定需要整块时间
重要的事情说三遍，请有辞职就能把副业做好想法的人不要来反驳我，认真的听进去我这句话。
很多人有执念，觉得自己的副业没做起来是因为自己没有投入足够多的时间，而没有投入足够多的时间是因为有主业影响了自己，所以总觉得要辞职去做副业副业才能起来。
但我告诉你一个残酷的事实，投入的时间虽然很重要，但绝不是副业成败的关键因素，核心是你对项目的认知和判断，你能不能抓住副业中的关键问题，然后找到解法，如果你抓住关键问题，可能并不需要你投入太多时间。
认知来源于哪？ 第一，先大量看相关内容，兵马未动粮草先行。做项目的粮草是什么？就是把所有相关内容都认真看几遍！
第二，看完相关内容后，你就会形成基本判断和基本的想法，抓住最关键的点，去做尝试和验证，这个过程可能会费点时间，但因为你可能已经抓住了关键点，所以比较容易形成一个非常清晰具体的问题或者困惑，而不是像现在这样在抽象层面打转，一旦有清晰具体的问题，在生财有术里面或者问人，大概率梦找到答案。
第三，以上第二步持续循环，你就能跑通从 0 到 1，并且已经开始赚到钱，这个时候你会认真去算账，你辞职出来后将你副业能放大到多少倍，与你主业收入相比如何，帐算清楚，答案自然就会出来。
所以，回到你的问题 1/ 我劝你不要辞职，不要让自己陷入到没有退路的阶段 2/ 你仍然可以在繁忙的主业里面抽一点点时间出来去琢磨副业 3/
接受副业进展慢的规律，但要保持日拱一卒的状态，迟早会有进展

欧欧｜心身探索 : 看了这个提问和大家的评论，真的很感慨，原来不止我一个人经历过这段艰难的时光。亦仁大大说的真是金玉良言。现在回头看，我也想结合自身的经历，分享一个可能被漏掉的视角：心理能量耗竭，也就是圈友写的：上班压力巨大很抗拒上班。我当年每天上班都是上坟的心情，感觉忙了一天啥有价值有意义的事也没干，但累的回家只想瘫倒，压根没办法做副业。
而心理能量耗竭，也正是因为圈友处在一种双重困境里，这是一个结构性的困境。
困境 1：如果我不改变（继续上班），我会继续被消耗、时间继续被锁死、我的生命就这样被浪费了；
困境 2：如果我改变（辞职做副业），我可能失败、会失去稳定收入、房贷养娃压力大、家庭可能有矛盾
结果：无论选哪条路，都感觉很难走通。这种双重困境会十分消耗心力，时间久了，就会产生心理能量/心力耗竭。而解决这个双重困境的方法，是需要找到属于你的第三条路，亦仁大大讲的“做副业并不一定需要整块时间，核心是你对项目的认知和判断”，本质上是改变对副业的认知，这也是第三条路的一种。
可当我们已经面临心理能量耗竭，就像你的“油箱”已经马上见底，是很难真正开往远方的。这时候首先需要做的其实是：休息一下，让心理能量恢复。就像你自己说的“这个状态真的很难受，想休息一段时间”，其实你知道自己现在最需要什么，而你现在做的“慢慢拉开和老板的距离，争取大礼包”都是很好的方法。我当年也用过类似的方法：
1、心理上罢工：承认现在的这份工作已经降级成了一份为了拿工资、交社保、等大礼包的生存工具。停止所有不必要的情感和精力投入。 2、建立防火墙：
让习惯追求上进追求成长的自己对这份工作，练习说“我不会”、“不归我管”、“我需要更多时间”。这不是不负责，这是在保护你那所剩不多比金子还宝贵的心力，不被无意义的事消耗。
3、规则允许下的物理隔离：试着问自己：我在公司制度允许内，能为自己争取到的、最长的一段合法休整期是什么？它可能是一次积攒已久的年假、一次（如果身体真的有信号）合规的病假、甚至是和领导申请一段时间的停薪留职（这个更适用于体制内）。
我当年就是靠一次长假，为自己赢得了喘息之机，去真正地思考我到底要什么。如果你在这个喘息的机会里，能够让自己好好休息当然最有利于心理能量的恢复，如果你还是着急，想多研究副业那也很好，等自己缓过来一些再来做选择，请一定相信，先照顾好自己，才是所有事业唯一的起点。
祝好~
