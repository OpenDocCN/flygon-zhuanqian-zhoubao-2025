## (34 赞)Claude code 实践：如何从 0 到 1 搭建公众号自动化工作流（附完整提示词）

作者： 周行今天学习了吗？

日期：2025-10-30

大家好，我是周行。 刚参加完生财的垂直小号内测航海，发现用 AI 写作时有个很大的痛点，就是。 用 Chat
GPT、Gemini 或豆包这些 AI 大模型，一次只能输出一篇，写多了还会同质化；用飞书+DeepSeek 或者 coze 倒是可以批量生产大量文案，但文章质量却很难得到保证。
所以，花了几天时间用搭建了一个。 这个工作流可以实现从选题到创作到排版的整个公众号创作流程。 并且根据你创作的领域和需求，可以做出灵活的调整。
这篇文章可能会有些长，适合有以下需求的朋友： 1.同时运用多个公众号，希望实现 AI 自动化创作，自己拿到文章就可以直接发布；
2.投入时间有限，想写公众号但又不想让 AI 水文； 3.了解用 Clauce code 创作内容的基础思路。 详见飞书👇： [`bcn6m62qmd6f.feishu.cn/wiki/BA4SwuXP6ik0m3ke2BIc6AMrnwh?from=from_copylink`](https://bcn6m62qmd6f.feishu.cn/wiki/BA4SwuXP6ik0m3ke2BIc6AMrnwh?from=from_copylink)
