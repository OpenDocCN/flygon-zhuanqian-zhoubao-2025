初创公司上班，下班后码字分享我的经验。 最近 Claude Skills 在 AI 圈火了，你可能在各种群里、朋友圈里看到有人在讨论它，说它是“比 MCP

Skills 真的能用吗？ 这篇文章，我会用一个真实的案例：“让 AI 自动帮我看 B 站测评视频，10 分钟生成选购指南”，完整还原我从 0 到 1 制作
