# 怎么分析的

## 主要通过 JTBD 和 第一性原理进行分析

JTBD = Jobs To Be Done

![](img/d81dbd61a3dac2b23d02e2ccdb1856b2.png)

https://hbr.org/2016/09/know-your-customers-jobs-to-be-done?ref=blog.helpdocs.io

## 分析的维度有

*   核心人群，分别站在消费者和生产者角度

*   需求侧本质需求分析，关键洞察

*   异常值

*   机会拆解

*   一句话总结，消费者和生产者
